🚀 AI Career Copilot - Resume Analyzer
An AI-powered web application that analyzes resumes and provides smart career insights like skills,
missing skills, roadmap, and interview questions.

---

📌 Features

- 📄 Resume analysis using AI  
- 🧠 Extracts key skills  
- 📉 Identifies missing skills  
- 🛣️ Generates career roadmap  
- 🎯 Provides interview questions  
- 🌐 Simple web interface  

---
 🛠️ Tech Stack

- Frontend: HTML, CSS  
- Backend: Python (Flask)  
- AI: OpenAI API  
- Database: SQLite
